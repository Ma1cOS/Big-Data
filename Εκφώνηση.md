# Big Data & Data Mining - Εξαμηνιαία Εργασία

## Στόχος Εργασίας

Υλοποίηση **end-to-end** συστήματος ανάλυσης δεδομένων, που θα περιλαμβάνει:

1. Data Preprocessing
2. Exploratory Data Analysis (EDA)
3. Εφαρμογή αλγορίθμων Data Mining
4. Αξιολόγηση μοντέλων
5. Εξαγωγή συμπερασμάτων

### Έννοιες που καλύπτει η εργασία

- Classification
- Neural Networks
- Naive Bayes
- SVM
- Association Rules
- Big Data Processing (Spark / MapReduce)

## Απαιτήσεις Εργασίας

### 1. Data Understanding & Preprocessing

- Περιγραφή dataset
- Cleaning (missing values, outliers)
- Feature engineering

### 2. Exploratory Data Analysis (EDA)

- Στατιστικά στοιχεία
- Visualizations (matplotlib / seaborn)

### 3. Εφαρμογή Αλγορίθμων (Υποχρεωτικά)

Κάθε ομάδα πρέπει να υλοποιήσει:

| Κατηγορία | Ποσότητα | Παραδείγματα |
|---|---|---|
| **Classification Models** | 2 | Decision Trees, SVM, Naive Bayes, Neural Networks |
| **Advanced Technique** | 1 | Association Rules, Clustering (k-means), Recommendation System, Graph Analysis (PageRank) |
| **Big Data Approach** | 1 (bonus) | Spark ή MapReduce (αν το dataset είναι μεγάλο) |

### 4. Model Evaluation

- Accuracy
- Precision / Recall
- Confusion Matrix
- ROC Curve (αν γίνεται)

### 5. Σύγκριση Μοντέλων

- Ποιο αποδίδει καλύτερα και γιατί

### 6. Συμπεράσματα

- Insights από τα δεδομένα
- Πρακτικές εφαρμογές
