Classification

Μοντέλα Μηχανικής Μάθησης

MLP
Νευρωνικό δίκτυο με μη-γραμμικά όρια απόφασης, backpropagation, πολλαπλά hidden layers
SVM
Linear SVC: αποδοτικό σε sparse δεδομένα, class weights για imbalance
Random Forest
Ensemble δέντρων με feature importance, robust σε υπερπροσαρμογή
Logistic Reg.
Baseline μοντέλο, γρήγορο, ερμηνεύσιμο, class_weight=balanced

Διαδικασία: 5-fold cross-validation + grid search υπέρ-παραμέτρων. Αξιολόγηση μόνο στο test set.

Αξιολόγηση

5 Μετρικές για Imbalanced Classification

Accuracy
% σωστών προβλέψεων. Παραπλανητική: 95% dummy
Precision
Πόσα predicted είναι όντως stroke. False alarms
Recall
Πόσα πραγματικά stroke βρήκαμε. Κλινικά κρίσιμο
F1-Score
Αρμονικός μέσος P & R. Ισορροπία
ROC-AUC
Διαχωριστική ικανότητα. Ανεξάρτητο από threshold

⚠ Αξιολόγηση αποκλειστικά στο test set (987, φυσικό imbalance 4.3%). Το test δεν συμμετέχει πουθενά στην εκπαίδευση.

Τεχνικές Αποφάσεις & Insights

  • Η ηλικία κυριαρχεί, ~25 έτη διαφορά, r=0.25
  • Class imbalance. To SMOTE διορθώνει, το test μένει ανέγγιχτο.
  • Μη-γραμμικές σχέσεις μεταξύ καπνίσματος, γλυκόζης, ηλικίας και υπέρτασης
  • Bronze → Silver → Gold: αναπαραγωγιμότητα, zero leakage
  • Test set ανέγγιχτο, 987 δείγματα χωρίς καμία επαφή με την εκπαίδευση
